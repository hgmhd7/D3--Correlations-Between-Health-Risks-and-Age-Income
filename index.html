U.S. Demographics Correlations

Correlations Between Health Risks, Age, Income, and Other Factors

This D3 visualization was created to take a deeper dive into different demographics across the U.S. to examine their correlations. The data set spans across all 50 states and explores six factors to see if we can find any links between them. Our dependents for the data are health, smoking, and obesity. Our independents for the set are poverty, age, and income. Please explore the data by clicking the different factors on the plot to see if you can spot any possible outliers, correlations, and anomalies. Which factors have the highest correlations? What states show the highest correlations overall? What states seem to look better overall in comparison? You can examine these and many more questions with this interactive plot.

The data was obtained from the United States Census Bureau. Visit the Fact Finder page on the US Bureau here. Also, check out my GitHub page that contains all of the original code for the project. Happy exploring!
